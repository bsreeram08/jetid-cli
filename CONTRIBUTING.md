# Contributing to jetid-cli

Thank you for your interest in improving `jetid-cli`!

## Development Setup

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/bsreeram08/jetid-cli.git
    cd jetid-cli
    ```

2.  **Install dependencies**:
    ```bash
    bun install
    ```

3.  **Run tests**:
    ```bash
    bun test
    ```

## Pull Request Process

1.  Create a new branch for your feature or bugfix.
2.  Ensure all tests pass.
3.  Add new tests for any new functionality.
4.  Update the README if you've added new flags or commands.
5.  Submit a PR with a clear description of the changes.

## Code Style

We use Bun's default formatting. Ensure your code is clean and follows the existing patterns in the codebase.
